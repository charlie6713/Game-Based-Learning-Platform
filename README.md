# Game-Based-Learning-Platform

init vite scaffold
by following code:
npm create vite@latest game-based-learning-platform -- --template react-ts

connect tailwindcss to our project
npm install tailwindcss @tailwindcss/vite
add import tailwindcss from '@tailwindcss/vite' in vite.config.ts file



cd game-based-learning-platform
npm install
npm run dev
